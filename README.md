# 🌤️ Weather App

A modern and responsive weather application built using **Vanilla JavaScript**, allowing users to search for any city and get real-time weather data with a clean UI.

---

## 🚀 Live Demo

👉 https://momenesam11.github.io/weather/

---

## 📌 Overview

This project is a simple weather application that fetches real-time data from an external API and displays:

* Current weather conditions
* Temperature
* Humidity
* Wind speed & direction
* 3-day forecast

The UI is designed to be clean, responsive, and user-friendly.

---

## 🧠 Features

* 🔍 Search for any city worldwide
* ⚡ Real-time weather data
* 📅 3-day forecast
* 🌡️ Temperature display in Celsius
* 💨 Wind speed & direction
* 💧 Humidity indicator
* 🎨 Smooth animations on data load
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript (Vanilla JS)
* Weather API

---

## 🔗 API Used

This project uses the Weather API:

* Base URL:
  https://api.weatherapi.com/v1

* Endpoint used:
  `/forecast.json`

> The app fetches weather data dynamically using `fetch()`
>

## ⚙️ How It Works

1. User enters a city name
2. The app sends a request to the API
3. Weather data is returned
4. Data is dynamically rendered in the UI

* Input field & UI structure defined in


* Styling handled using custom CSS
